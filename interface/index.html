Mild Learning

マイルドヤンキー風歌詞自動生成ツール


Technology

実際の歌詞から学習したニューラルネット言語モデルを使い、歌詞の文を自動で生成します。生成した文を、意味内容を考慮した分類器により曲として構成します。

新規性 Novelty
機械学習のフロンティアとして注目を浴びているDeep Learningを日本語歌詞生成にアレンジした新しい試みです
This is a novel challenge to utilize Deep Learning, which has been drawning much attention, to generate Japanese lyrics
技術性 Technology
Long-Short Term Memory (LSTM) と呼ばれる高度なニューラルネットワークモデルを使用しているほか、Feed-Forward型ニューラルネットワークを意味解析のために使用しています
This system consists of Long-Short Term Memory (LSTM), a sophisticated neural network-based model, as well as Feed-Forward neural network to handle semantics
発展性 Prospective
このシステムは人間の発想では思いつかないようなアイデアを提供することで、エンターテイメント業界に新しい音楽の可能性をもたらします。また、歌詞生成に使うコンポーネントはあらゆるテキストから学習可能で、例えばその日の全ニュースをもとに歌詞を作ることも可能です
This system has a potential impact on music entertainment, as it is able to create vivid ideas that we cannot come up with. In addition, any kinds of texts can be used as input for the system, and we can make a song based on all news on that day for example
再現性 Reproducibility
このシステムを運用するためには多くのDeep Learning手法と同じくGPU計算環境が必要ですが、データ準備は非常に簡単で、労力を要しません
This system requires GPU machines as other Deep Learning methods do. On the other hand, we do not need many efforts to prepare datasets and do pre-processing or post-processing
