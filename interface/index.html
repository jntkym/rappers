Mild Learning

マイルドヤンキー風歌詞自動生成ツール


Technology

実際の歌詞から学習したニューラルネット言語モデルを使い、歌詞の文を自動で生成します。生成した文を、意味内容を考慮した分類器により曲として構成します。

新規性 Novelty
機械学習のフロンティアとして注目を浴びているDeep Learningを日本語歌詞生成にアレンジした新しい試みです
This system adopts a novel approach to utilize Deep Learning techniques to generate Japanese lyrics
技術性 Technology
Long-Short Term Memory (LSTM) と呼ばれる高度なニューラルネットワークモデルを使用しているほか、Feed-Forward型ニューラルネットワークを意味解析のために使用しています
In order to handle semantics the system uses a tandem of two neural networks - Long-Short Term Memory (LSTM) and Feed-Forward (FF)
発展性 Prospective
このシステムは人間の発想では思いつかないようなアイデアを提供することで、エンターテイメント業界に新しい音楽の可能性をもたらします。また、歌詞生成に使うコンポーネントはあらゆるテキストから学習可能で、例えばその日の全ニュースをもとに歌詞を作ることも可能です
This system has a potentially large impact on music industry, as it is able to generate new vivid ideas laying beyond human imaginative abilities. Moreover, any kinds of texts can be used as an input for the system - for example, we can create songs based on the latest news records
再現性 Reproducibility
このシステムを運用するためには多くのDeep Learning手法と同じくGPU計算環境が必要ですが、データ準備は非常に簡単で、労力を要しません
Just like other Deep Learning techniques, our system requires GPU machines for faster training. On the other hand, we do not need to put much effort into preparing datasets and doing pre- or post-processing
